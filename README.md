## 👨‍💻 Über mich

Als **Full Stack Engineer** entwickle ich skalierbare Anwendungen an der Schnittstelle zwischen komplexer Business-Logik und benutzerorientierten Interfaces. Mein Fokus liegt auf:

- **Backend-Architektur**: Robuste, wartbare Systeme mit C#, Java und modernen Frameworks
- **Enterprise-Lösungen**: SQL Server, verteilte Systeme und Datenbankoptimierung
- **Frontend-Excellence**: Responsive, performante Web-Applikationen mit JavaScript/TypeScript
- **Infrastructure & DevOps**: Proxmox, Virtualisierung und moderne Deployment-Strategien

Ich arbeite leidenschaftlich an Projekten, die echte Probleme lösen und mich technical pushes geben. Mein Anspruch ist es, nicht nur Code zu schreiben, sondern nachhaltige Architekturen zu bauen.
